# AMD Interactive Session

An immersive gaming-style questionnaire application built for project `amd-first-494006`.

## Features
- **Gaming HUD**: Immersive digital interface with scanlines and glitch effects.
- **Project Integration**: HUD displays Project ID `amd-first-494006` and Project Number `1035333686490`.
- **Interactive Session**: Diagnostic-style questionnaire with typewriter text and real-time timer.
- **Session Analysis**: Provides a final rank (e.g., ELITE-X) based on session completion.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Fonts: Orbitron & JetBrains Mono

## How to Use
1. Open `index.html`.
2. Click **BEGIN INTERFACE** to start the diagnostic sequence.
3. Select answers for each system prompt.
4. Review your session analysis and final rank.
